# Portfolio Amand4May

Portfolio pessoal em HTML estatico, com duas versoes da pagina:

- `index.html` em portugues
- `En-index.html` em ingles

O projeto tem um visual neon em rosa, cards com borda, efeito de reveal ao rolar a pagina e uma secao de projetos com cards clicaveis, modal e galeria de imagens.

## Destaques

- Hero principal com foto, nome e links sociais
- Secao sobre mim
- Secao de projetos com fotos reais
- Pop-up com imagem ampliada, miniaturas e link do repositorio
- Secao de curriculo com download em PDF

## Tecnologias

- HTML
- Tailwind CSS via CDN
- JavaScript puro
- Google Fonts (`Work Sans`)

## Estrutura

- `index.html` - versao PT
- `En-index.html` - versao EN
- `pics/` - imagens dos projetos e foto de perfil
- `curriculo.pdf` - curriculo para download

## Como abrir

1. Abra a pasta do projeto no VS Code.
2. Abra `index.html` ou `En-index.html` no navegador.
3. Se preferir, use uma extensao Live Server para ver as alteracoes em tempo real.

## Observacoes

- Os cards de projeto usam imagens da pasta `pics/`.
- O modal mostra a imagem principal em tamanho maior e miniaturas adicionais.
- Os links de repositorio podem ser editados diretamente no arquivo HTML de cada idioma.
